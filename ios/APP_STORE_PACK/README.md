# App Store Submission Pack

This folder contains all materials needed for App Store submission.

## Contents

### App Description & Metadata

- `app_description.txt` - Full app description (up to 4000 characters)
- `app_subtitle.txt` - App subtitle (30 characters max)
- `keywords.txt` - Keywords (100 characters max, comma-separated)
- `promo_text.txt` - Promotional text (170 characters max, can be updated without new version)
- `release_notes_template.txt` - Template for version release notes

### Privacy & Support

- `privacy_policy_template.md` - Privacy policy template (update with actual URLs)
- `support_url.txt` - Support URL placeholder
- `marketing_url.txt` - Marketing URL placeholder

### Age Rating

- `age_rating_questionnaire.md` - Answers to App Store age rating questions

### Screenshots

See `screenshot_shot_list.md` for required screenshots and how to capture them.

### Demo Script

See `demo_script.md` for a walkthrough suitable for App Store reviewers.
