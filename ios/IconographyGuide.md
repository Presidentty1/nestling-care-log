# Iconography Guide

- Feed: `drop.fill` (bottle) — pair with text “Feed”
- Sleep: `moon.fill` / `moon.zzz.fill` (active) — text “Sleep” or “Stop Sleep”
- Diaper: `drop.circle.fill` — text “Diaper”
- Tummy time: `figure.child` — text “Tummy”
- Add: `plus.circle.fill` only for single add action (never multi-action)
- Timeline menu: `ellipsis` (more)
- Success: `checkmark.circle.fill`, Error: `xmark.circle.fill`, Info: `info.circle.fill`
- Navigation close/done: `xmark` for sheets, `chevron.left` for back

Rules

- Always pair icons with text labels.
- Avoid generic `plus` for multi-type actions; use distinct icons per type.
- Keep touch targets ≥44pt and ensure color contrast for icons on colored fills.
