# UX Testing Checklist

- [ ] First-time user: empty state → quick log → see timeline
- [ ] Undo delete within 5s toast
- [ ] Edit from timeline swipe and menu
- [ ] Feature tooltip appears after 3+ logs and can be dismissed
- [ ] Adaptive layout on Plus/Max landscape (status tiles + quick actions)
- [ ] VoiceOver reads quick actions, status tiles, toasts, empty states
- [ ] Dynamic Type at largest size keeps buttons tappable (≥44pt)
- [ ] Dark mode contrast for cards, text, icons
- [ ] Haptics: light for selection, medium for save, heavy for destructive
- [ ] Sheets and modals present consistent Done/Close controls
- [ ] Offline/slow network states show skeletons or loading indicators
- [ ] Tab navigation returns to Home without getting stuck in modals
